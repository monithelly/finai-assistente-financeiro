# 💰 FinAI — Assistente Financeiro Inteligente

O **FinAI** é uma aplicação interativa desenvolvida em Streamlit que simula um assistente financeiro, permitindo ao usuário tirar dúvidas, entender conceitos e realizar simulações de forma simples e intuitiva.

O projeto aplica conceitos de:
- Inteligência artificial (simulação de comportamento conversacional)
- Experiência do usuário (UX)
- Lógica de programação em Python
- Educação financeira

## 🚀 Funcionalidades

## ▶️ Como executar o projeto

1. Clone o repositório:

git clone https://github.com/seu-usuario/finai-assistente-financeiro.git

2. Acesse a pasta:

cd finai-assistente-financeiro

3. Instale as dependências:

pip install -r requirements.txt

4. Execute a aplicação:

streamlit run app.py

---

## ⚠️ Observação 

Este projeto está totalmente funcional, porém a execução completa depende de créditos ativos na API da OpenAI.

Caso não haja créditos disponíveis, a chamada para geração de resposta pode retornar erro de limite (`insufficient_quota`).

O código e a arquitetura foram testados e validados até a etapa de integração com a API.
